## 2. 진행 방식 
---
1. 스터디 날까지 각자 해당 주차 강의를 공부한다. 
2. 강의를 들으며 공유할 부분, 질문에 대해 정리해온다.
3. 스터디 다음날 자정까지 각자 공부한 내용을 문서화해 공유한다.
